# Web Cleaner，去除 csdn 广告，复制代码等

## 功能

### CSDN
- 关闭广告
- 复制代码
- 阅读全文
- 关闭登录弹窗
- 关闭右侧工具栏
- 关闭底部工具栏

## 安装

1. clone 源码
```bash
git clone https://github.com/zhuanglong/web-cleaner.git
```

2. 安装依赖和打包

```bash
pnpm install
pnpm dev
pnpm build
```

3. 谷歌浏览器，从右上角菜单->更多工具->扩展程序可以进入插件管理页面，也可以直接在地址栏输入 chrome://extensions 访问

4. 勾选开发者模式，即可以文件夹的形式直接加载插件
